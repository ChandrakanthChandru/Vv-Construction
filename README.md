# VV Construction & Interior Design — Website

## Folder Structure
```
vv-construction/
├── index.html          ← Home page
├── about.html          ← About page
├── services.html       ← Services page
├── projects.html       ← Projects / Gallery page
├── contact.html        ← Contact page
├── css/
│   └── style.css       ← All styles
├── js/
│   └── script.js       ← All JavaScript
├── images/             ← Add your own images here
└── assets/             ← Optional local libraries
```

## How to Run Locally
1. Extract the zip into any folder
2. Open index.html in your browser — that's it!
3. No server required. All CDN links load externally.

## Customise
- Replace +919XXXXXXXXX with your actual WhatsApp / phone number (Ctrl+H in any editor)
- Replace placeholder address / email in footer and contact page
- Add your actual Google Maps embed URL in contact.html
- Drop real project photos into /images and update src paths

## Tech Stack
- HTML5, CSS3, Vanilla JS
- Google Fonts: Playfair Display + Outfit
- Font Awesome 6.5 (CDN)
- AOS 2.3.4 Animate On Scroll (CDN)
